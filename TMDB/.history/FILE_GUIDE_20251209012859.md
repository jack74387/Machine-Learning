## 🚫 常見錯誤

### 錯誤 1: 使用了 main-2.py

**問題**: 複製貼上時出現語法錯誤
**解決**: 改用 `kaggle_notebook.py`

### 錯誤 2: 路徑設定錯誤

**問題**: FileNotFoundError
**解決**: 
- Kaggle: 使用 `kaggle_notebook.py`（已設定正確路徑）
- 本地: 確保資料檔案在同一目錄

### 錯誤 3: 套件未安裝

**問題**: ModuleNotFoundError
**解決**: 
- 本地: `pip install -r requirements.txt`
- Kaggle: 使用 `kaggle_notebook.py`（自動檢測）

### 錯誤 4: 不知道用哪個檔案

**問題**: 檔案太多，不知道選哪個
**解決**: 
- Kaggle 提交 → `kaggle_notebook.py`
- 本地執行 → `run_complete_pipeline.py`
- 快速測試 → `simple_main.py`

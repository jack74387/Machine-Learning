# TMDB 電影票房預測專案 - Taskmaster 任務規劃

## 專案概述
使用 TMDB (The Movie Database) 電影資料集預測電影票房收入

**資料來源**: https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata

**目標**: 預測電影的 revenue (票房收入)

---

## 任務分解 (Taskmaster Method)

### Phase 1: 問題定義與資料理解
- [x] 確認問題類型：回歸問題 (預測連續值 revenue)
- [ ] 載入並檢視資料集
- [ ] 分析資料欄位與特徵
- [ ] 檢查資料品質（缺失值、異常值）

### Phase 2: 探索性資料分析 (EDA)
- [ ] 分析目標變數 revenue 的分布
- [ ] 分析數值型特徵（budget, popularity, runtime）
- [ ] 分析類別型特徵（genres, cast, crew, keywords）
- [ ] 特徵相關性分析
- [ ] 視覺化關鍵發現

### Phase 3: 資料預處理
- [ ] 處理缺失值
- [ ] 處理 JSON 格式欄位（genres, cast, crew, keywords 等）
- [ ] 特徵工程
  - [ ] 從 release_date 提取時間特徵
  - [ ] 從 cast/crew 提取關鍵人物
  - [ ] 從 genres 提取類型特徵
  - [ ] 創建衍生特徵（如 budget_popularity_ratio）
- [ ] 特徵編碼與標準化
- [ ] 分割訓練集與驗證集

### Phase 4: 模型建立與訓練
- [ ] Baseline 模型（簡單線性回歸）
- [ ] 進階模型
  - [ ] Random Forest
  - [ ] XGBoost
  - [ ] LightGBM
  - [ ] CatBoost
- [ ] 超參數調優
- [ ] 交叉驗證

### Phase 5: 模型融合 (Voting/Ensemble)
- [ ] 實作 Voting Regressor
- [ ] 實作 Stacking
- [ ] 實作 Blending
- [ ] 比較不同融合策略

### Phase 6: 評估與優化
- [ ] 評估指標：RMSE, MAE, R²
- [ ] 特徵重要性分析
- [ ] 錯誤分析
- [ ] 模型優化

### Phase 7: 預測與提交
- [ ] 對測試集進行預測
- [ ] 生成提交檔案
- [ ] 結果分析

---

## 評估指標
- **主要指標**: RMSE (Root Mean Squared Error)
- **次要指標**: MAE (Mean Absolute Error), R² Score

- xgboost, lightgbm, catboost

## 備註
- 資料集包含大量 JSON 格式的文本欄位，需要特別處理
- 考慮使用 log transformation 處理 revenue 的偏態分布
- 注意處理 budget=0 的情況
